# ADR-0009: Project name "Corveon"

- **Status:** Accepted
- **Date:** 2026-07-07

## Context
The platform needed a coined, collision-free, brandable name that does not narrowly describe one
module (medication safety is one module, not the whole product).

## Decision
Adopt **Corveon** — evoking *cor* (Latin "heart/core": the clinical core and central orchestrator)
with a modern *-eon* suffix (durability/scale). Short, globally pronounceable, no negative
connotations in major languages.

## Consequences
- A July 2026 collision check found no healthcare/AI/medical company, product, GitHub org, or obvious
  trademark using "Corveon" — only unrelated personal social handles.
- Phonetic neighbors exist in health-AI (Corva Health, Corti, CorroHealth), so a formal USPTO/WIPO
  **trademark clearance is advised before commercial launch**. Safe to adopt now.

## Alternatives considered
Each a verified collision and therefore rejected: **Medrona** (active RCM healthcare company),
**Veritome** (EU AI Act compliance vendor), **Evidium** (funded healthcare-AI startup),
**Aletheia** (active healthcare data/marketing firms).
